Description détaillée

Cette classe permet de se connecter et d'envoyer des requetes sur les DB Oracle.

Cette classe permet de centraliser les variables de connexion et de faire abstraction de la librairie utilisée pour la connection (ici OCI).

recuperer_decision_par_numero_de_decision ( $  numero_de_decision)

Permet de récupérer une demande dans la DB Oracle sur base de son numéro de décision 400. Les dates date_du, date_au et date de séance sont castée au format YYMMDD pour faciliter les comparaisons avec le 400.

Paramètres:
$numero_de_decisionLe numdec de la décision dans le 400.
Renvoie:
Un tableau contenant les index :
  • ["resultat"] = un array contenant tous les records renvoyées.
  • ["erreur"] = Booléen signalant si une erreur a été rencontrée.
  • ["message"] = Un éventuel message, par exemple pour décrire l'erreur rencontrée.
recuperer_tableau_requete ( $  requete,
$  mode = 5 
)

Permet de récupérer un tableau html du résultat de la requete.

Paramètres:
$requeteLa requete SQL
$modeLe mode OCI de la requete, par défaut 5 = OCI_ASSOC + OCI_RETURN_NULLS
Renvoie:
Un tableau contenant les index :
  • ["tableau"] = le code html du tableau représentant les résultats.
  • ["resultat"] = un array contenant tous les records renvoyées.
  • ["erreur"] = Booléen signalant si une erreur a été rencontrée.
  • ["message"] = Un éventuel message, par exemple pour décrire l'erreur rencontrée.
