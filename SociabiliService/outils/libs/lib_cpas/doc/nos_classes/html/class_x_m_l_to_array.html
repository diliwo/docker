Description détaillée

Cette classe permet de manipuler et d'afficher plus aisément les chaines de caractères XML.

Grâce à cette classe, on peut retrouver la valeurs de chaque balise d'un nom spécifique, ou qu'elle se trouve dans l'arborescence du XML. On peut également récupérer la chaine XML sous forme d'un tableau PHP.
